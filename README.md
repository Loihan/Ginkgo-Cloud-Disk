# 银杏网盘 (Ginkgo Cloud) - 一个功能丰富的个人云存储 Web 应用

这是一个使用 Vue 3 和 Node.js (Express) 技术栈构建的全功能个人网盘项目。用户可以注册、登录、上传/下载文件、管理文件夹、创建公开分享链接，并查看自己的存储空间使用情况。

## ✨ 主要功能

- **用户系统**:
  - [x] 用户注册与登录 (密码使用 bcrypt 加密)
  - [x] JWT (JSON Web Token) 认证，保障接口安全
  - [x] 支持修改用户名和上传个人头像

- **文件与文件夹管理**:
  - [x] 创建文件夹
  - [x] 文件/文件夹的重命名
  - [x] 拖拽或点击上传文件 (最大 60MB)
  - [x] 文件下载
  - [x] 批量移动文件/文件夹到任意目录
  - [x] 批量打包文件/文件夹到当前目录下的新文件夹中
  - [x] 右键上下文菜单，提供快捷操作

- **高级功能**:
  - [x] **回收站**: “软删除”机制，支持从回收站还原或彻底删除文件。
  - [x] **高级分享**:
    - 支持文件和文件夹分享
    - 可为分享链接设置**访问密码**
    - 可设置分享链接的**有效期** (1天, 7天, 永久)
  - [x] **分享管理**: 独立的“我的分享”页面，可查看、复制和取消分享。
  - [x] **全局搜索**: 在整个网盘范围内按文件名模糊搜索。
  - [x] **空间配额**:
    - 每个用户拥有 1GB 的存储空间。
    - 独立的“我的空间”视图，通过多彩环形图和列表展示各类文件的空间占用情况。
  - [x] **传输列表**:
    - 实时显示上传进度。
    - 记录上传和下载历史。
    - 支持按用户隔离历史记录，并可选择性删除。

## 🛠️ 技术栈

### 前端
- **框架**: Vue 3 (Composition API, `<script setup>`)
- **UI 库**: Element Plus
- **路由**: Vue Router
- **状态管理**: Pinia
- **HTTP 请求**: Axios

### 后端
- **框架**: Node.js + Express
- **数据库**: MySQL
- **ORM**: `mysql2/promise`
- **文件上传**: Multer
- **认证**: JSONWebToken (JWT)
- **密码加密**: Bcrypt

## 🚀 如何运行

### 前提条件
- 已安装 Node.js (v16+)
- 已安装 MySQL 数据库

### 后端设置
1.  **克隆项目**: `git clone [你的仓库地址]`
2.  **进入后端目录**: `cd [后端项目文件夹]`
3.  **安装依赖**: `npm install`
4.  **数据库配置**:
    - 创建一个名为 `网盘` 的数据库。
    - 根据项目中的 SQL 文件或命令，创建 `user_info`, `files`, `shares` 表。
    - 修改 `express_index.js` 文件中的 `pool` 配置，填入您自己的 MySQL 用户名和密码。
5.  **运行后端服务**: `node express_index.js`
    - 服务将运行在 `http://localhost:8082`

### 前端设置
1.  **进入前端目录**: `cd [前端项目文件夹]`
2.  **安装依赖**: `npm install`
3.  **API 地址配置**:
    - 打开 `src/api/index.js` 文件。
    - 修改 `baseURL` 为您的后端服务器地址 (例如 `http://localhost:8082`)。
4.  **运行前端开发服务**: `npm run serve`
    - 服务将运行在 `http://localhost:8080` (或您配置的其他端口)。
